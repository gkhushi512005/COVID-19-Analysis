## COVID-19 Data Analysis & Trend Visualization

## Overview

Analyzed global COVID-19 data using Python and performed time-series analysis to identify infection trends, compare country-wise pandemic impact, and visualize the global spread of COVID-19 through interactive dashboards and maps.

## Tools Used

* Python
* Pandas
* NumPy
* Plotly
* Matplotlib
* Jupyter Notebook
* Git & GitHub

## Dataset

**Our World in Data COVID-19 Dataset**

Dataset Link:
https://covid.ourworldindata.org/data/owid-covid-data.csv

Source:
https://ourworldindata.org/covid-cases

## Features

* Data cleaning and preprocessing
* Global COVID-19 trend analysis
* Country-wise infection trend comparison
* Top affected countries analysis
* Interactive choropleth world map visualization
* Time-series analysis of daily COVID-19 cases

## Key Insights

* The United States recorded the highest cumulative number of COVID-19 cases among all countries in the dataset.
* India experienced major infection waves, with a significant surge during 2021.
* Several countries showed recurring infection waves, highlighting the long-term impact of the pandemic.
* Global case counts increased rapidly during major variant outbreaks and gradually declined afterward.
* Vaccination campaigns were followed by a noticeable reduction in severe outbreaks and mortality rates in many regions.

## Conclusion

This project demonstrates how data analytics and visualization techniques can be used to understand the spread and impact of COVID-19 across different countries. The analysis highlights important pandemic trends and provides meaningful insights through interactive visualizations.
